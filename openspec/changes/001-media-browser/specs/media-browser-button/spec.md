# Capability: media-browser-button

**Status:** new
**Owner:** Multigram iOS app
**Persona:** end_user

## Description

Кнопка быстрого доступа к медиабраузеру в navigation bar экрана чата. Располагается между именем собеседника и аватаром. Доступна только в 1-на-1 чатах.

---

## Functional Requirements

### Requirement: Отображение кнопки в 1-на-1 чате

Кнопка медиабраузера SHALL отображаться в navigation bar экрана чата между блоком имени/статуса и аватаром собеседника. Кнопка представляет собой круглый зелёный элемент.

#### Scenario: Кнопка видна при открытии чата

```
GIVEN пользователь открыл 1-на-1 чат с собеседником
WHEN экран чата полностью загрузился
THEN в navigation bar между именем и аватаром отображается круглая зелёная кнопка медиабраузера
```

#### Scenario: Кнопка скрыта в групповых чатах

```
GIVEN пользователь открыл групповой чат или канал
WHEN экран чата полностью загрузился
THEN кнопка медиабраузера НЕ отображается в navigation bar
```

### Requirement: Открытие медиабраузера по тапу

При нажатии на кнопку SHALL открываться модальное окно медиабраузера поверх текущего чата.

#### Scenario: Тап открывает модалку

```
GIVEN пользователь находится в 1-на-1 чате
  AND кнопка медиабраузера отображается
WHEN пользователь нажимает на кнопку
THEN открывается модальное окно медиабраузера (media-browser-modal)
  AND чат остаётся виден под модальным окном
```

### Requirement: Кнопка не мешает существующим элементам

Кнопка SHALL НЕ перекрывать и не сдвигать существующие элементы navigation bar (кнопка "Back", имя, статус, аватар, кнопка звонка).

#### Scenario: Layout navigation bar сохраняется

```
GIVEN пользователь открыл 1-на-1 чат
WHEN экран чата отображается с кнопкой медиабраузера
THEN кнопка "< Chat" отображается слева
  AND имя и статус собеседника отображаются по центру
  AND кнопка медиабраузера отображается между именем и аватаром
  AND аватар собеседника отображается справа
  AND ни один элемент не обрезан и не перекрыт
```

---

## Non-Functional Requirements

- **Отзывчивость:** tap-to-modal transition SHALL занимать < 200ms (p99)
- **Размер кнопки:** минимальная область тапа 44x44pt (Apple HIG)
- **Адаптивность:** кнопка SHALL корректно отображаться на всех размерах экранов iPhone

## Service Boundaries

- **Владелец:** Multigram iOS app, модуль ChatController
- **Runtime:** Swift (iOS)
- **Зависимости:**
  - ChatControllerNode (hard) — точка интеграции кнопки
  - media-browser-modal (hard) — открывается по тапу

## Error Conditions

| Код | Описание | Поведение |
|-----|----------|-----------|
