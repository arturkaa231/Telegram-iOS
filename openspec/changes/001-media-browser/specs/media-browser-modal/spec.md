# Capability: media-browser-modal

**Status:** new
**Owner:** Multigram iOS app
**Persona:** end_user

## Description

Модальное окно медиабраузера, отображающее все медиафайлы текущего чата. Содержит заглушку видеоплеера в верхней части, горизонтальные табы фильтрации и список файлов с ленивой подгрузкой. Открывается поверх чата.

---

## Functional Requirements

### Requirement: Модальное окно поверх чата

Медиабраузер SHALL открываться как модальное окно поверх экрана чата, не скрывая чат полностью.

#### Scenario: Модалка открывается с анимацией

```
GIVEN пользователь нажал кнопку медиабраузера в чате
WHEN модальное окно открывается
THEN окно анимированно появляется снизу вверх
  AND чат остаётся частично виден за модальным окном
  AND navigation bar чата остаётся видимым
```

#### Scenario: Закрытие модалки

```
GIVEN модальное окно медиабраузера открыто
WHEN пользователь свайпает модалку вниз ИЛИ нажимает кнопку "назад"
THEN модалка закрывается с анимацией
  AND пользователь возвращается в чат
```

### Requirement: Заглушка видеоплеера

В верхней части модального окна SHALL отображаться область-заглушка видеоплеера.

#### Scenario: Отображение заглушки

```
GIVEN модальное окно медиабраузера открыто
WHEN окно полностью отрисовано
THEN в верхней части отображается прямоугольная область-заглушка
  AND заглушка имеет тёмный фон и placeholder-контент
  AND заглушка занимает ~40% высоты экрана
```

### Requirement: Табы фильтрации медиа

Под заглушкой видеоплеера SHALL располагаться горизонтально скроллируемая панель табов.

#### Scenario: Отображение табов

```
GIVEN модальное окно медиабраузера открыто
WHEN панель табов отрисована
THEN отображаются табы: "На телике", "Закреп", "Все файлы" и другие
  AND табы горизонтально скроллируются при нехватке ширины
  AND активный таб визуально выделен (подсветка фона)
```

#### Scenario: Переключение табов

```
GIVEN модальное окно открыто, активен таб "Все файлы"
WHEN пользователь нажимает на таб "Закреп"
THEN таб "Закреп" становится активным
  AND список медиа обновляется — отображаются только закреплённые файлы
  AND предыдущий таб теряет выделение
```

### Requirement: Список медиафайлов

Под табами SHALL отображаться вертикальный список медиафайлов чата.

#### Scenario: Отображение элемента списка

```
GIVEN модальное окно открыто, список файлов загружен
WHEN элемент списка отрисовывается
THEN каждый элемент отображает:
  - Превью/иконку типа файла (слева)
  - Имя файла
  - Имя отправителя
  - Время отправки
  - Размер файла
```

#### Scenario: Пустое состояние

```
GIVEN пользователь открыл медиабраузер в чате без медиафайлов
WHEN список загрузился
THEN отображается сообщение "Нет медиафайлов"
  AND табы всё равно отображаются
```

### Requirement: Кнопка "назад" в табах

Панель табов SHALL содержать кнопку "назад" (стрелка влево) слева от табов.

#### Scenario: Нажатие кнопки назад

```
GIVEN модальное окно медиабраузера открыто
WHEN пользователь нажимает кнопку "назад" на панели табов
THEN модальное окно закрывается
  AND пользователь возвращается в чат
```

---

## Non-Functional Requirements

- **Первая отрисовка:** модальное окно SHALL отображаться < 300ms после тапа (p99)
- **Анимация:** open/close анимация 250ms (UIKit spring animation)
- **Память:** модалка SHALL НЕ увеличивать memory footprint > 50MB при 100 файлах в списке
- **Жесты:** dismiss gesture SHALL работать при свайпе вниз > 100pt

## Service Boundaries

- **Владелец:** Multigram iOS app, новый модуль MediaBrowser
- **Runtime:** Swift (iOS)
- **Зависимости:**
  - media-browser-button (hard) — точка входа
  - media-lazy-loading (hard) — подгрузка данных
  - TelegramCore shared media API (hard) — источник данных
  - ChatController (soft) — контекст чата (peerId)

## Data Model

| Поле | Тип | Описание |
|------|-----|----------|
| peerId | EnginePeer.Id | ID собеседника |
| mediaItems | [MediaItem] | Загруженные элементы |
| activeTab | TabFilter | Текущий активный таб |
| isLoading | Bool | Идёт загрузка данных |

**TabFilter enum:** `onTV`, `pinned`, `allFiles`, `video`, `photo`, `audio`

**MediaItem:**

| Поле | Тип | Описание |
|------|-----|----------|
| messageId | EngineMessage.Id | ID сообщения |
| fileName | String | Имя файла |
| senderName | String | Имя отправителя |
| timestamp | Date | Время отправки |
| fileSize | Int64 | Размер в байтах |
| mediaType | MediaType | Тип медиа (photo/video/file/audio) |
| thumbnailResource | EngineMediaResource? | Ресурс превью |

## Error Conditions

| Код | Описание | Поведение |
|-----|----------|-----------|
| NETWORK_ERROR | Ошибка сети при загрузке медиа | Показать "Ошибка загрузки" + кнопка "Повторить" |
| EMPTY_CHAT | Чат без медиафайлов | Показать пустое состояние |
| TAB_NO_RESULTS | Нет файлов для выбранного таба | Показать "Нет файлов" в области списка |
