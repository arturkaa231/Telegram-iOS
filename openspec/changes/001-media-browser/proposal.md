# Proposal: Media Browser Button

**Change ID:** 001-media-browser
**Date:** 2026-05-13
**Author:** Aleksey Osipov

## Why

В стандартном Telegram для просмотра медиафайлов чата нужно зайти в профиль собеседника, найти раздел "Shared Media", переключиться на нужный таб. Это 3-4 тапа и потеря контекста разговора. При частой работе с файлами в чате (рабочие переписки, обмен документами/видео) это создаёт трение — пользователь хочет быстро найти нужный файл, не покидая чат.

## What Changes

- **NEW**: Кнопка медиабраузера в navigation bar чата (между именем контакта и аватаром)
- **NEW**: Модальное окно медиабраузера поверх чата с табами (фильтры по типу медиа)
- **NEW**: Ленивая подгрузка медиафайлов с бесконечной прокруткой
- **NEW**: Заглушка видеоплеера в верхней части модального окна (без воспроизведения в v1)

## Capabilities

### New Capabilities

- **media-browser-button** — кнопка в navigation bar экрана чата, открывающая медиабраузер
- **media-browser-modal** — модальное окно поверх чата с табами и списком медиафайлов
- **media-lazy-loading** — ленивая подгрузка и бесконечная прокрутка списка медиа чата

### Modified Capabilities

_Нет. Существующий UI чата не меняется — кнопка добавляется в свободное место navigation bar._

## Impact

- **UI**: Navigation bar экрана чата (ChatController) — добавление кнопки
- **UI**: Новый модальный контроллер поверх чата
- **Data**: Используем существующий API Telegram для получения shared media (searchMessages с фильтрами по типу)
- **Submodules**: Новый модуль в `submodules/TelegramUI/Components/` для медиабраузера
- **Upstream**: Минимальные изменения — только точка интеграции кнопки в ChatController

## Success Preview

- Доступ к медиа чата в 1 тап (было 3-4)
- Модальное окно не теряет контекст чата (чат виден под модалкой)
- Первая порция медиа (20 элементов) загружается < 1с

## Research Notes

**Домен:** iOS UI, Telegram shared media API.

**Ограничения платформы:**
- Navigation bar в Telegram iOS кастомный (не UINavigationBar) — нужно изучить ChatControllerNode для точки интеграции
- Shared media API уже есть в TelegramCore (searchMessages с mediaFilter) — новый backend не нужен
- Модальное окно должно учитывать safe area и жесты dismiss

**Domain Glossary:**
- **Media Browser** — модальный UI для просмотра всех медиафайлов чата
- **Shared Media** — медиафайлы, отправленные в чате (фото, видео, файлы, аудио)
- **Lazy Loading** — подгрузка следующей порции медиа при приближении к концу списка
- **Navigation Bar Button** — кнопка в верхней панели экрана чата

**Решения (из обсуждения):**
- Табы: "На телике", "Закреп", "Все файлы" + другие как на макете — нужны в v1
- Видеоплеер: заглушка в v1 (placeholder без воспроизведения)
- Групповые чаты: кнопка НЕ показывается, только 1-на-1 чаты
